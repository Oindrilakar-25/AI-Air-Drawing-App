# 🎨 AI Air Drawing App

An AI-powered virtual drawing application that uses **MediaPipe Hand Tracking** to enable touchless drawing with real-time gesture recognition. Draw in the air using your index finger, erase using hand gestures, and customize your brush with neon glow effects.

## 🚀 Features

- ✋ Real-time hand tracking using MediaPipe
- 🎨 Touchless air drawing
- 🌈 Customizable brush color
- 🖌️ Adjustable brush size
- ✨ Neon glow drawing effect
- 🧽 Gesture-based eraser
- ⏸️ Pause drawing with hand gestures
- 📷 Live webcam integration
- 💻 Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- MediaPipe Hands
- Camera Utils API

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Oindrilakar-25/AI-Air-Drawing-App.git
```

2. Open the project folder.

3. Run the project using **Live Server** in Visual Studio Code.

4. Allow camera permission when prompted.

5. Start drawing in the air!

## ✋ Gesture Controls

| Gesture | Action |
|---------|--------|
| ☝️ Index Finger | Draw |
| 🖐️ Open Palm | Erase |
| ✊ Closed Hand | Pause |
